## Application Overview
MINDSETHAPPYBOT is a Telegram bot for positive thinking development:
- Asks users "What good happened?" every 3 hours
- Stores responses in PostgreSQL with pgvector
- Uses OpenAI GPT-4 for personalized responses
- Uses Whisper API for voice message transcription
- Admin panel for monitoring users and messages

## Tech Stack
- Backend: Python 3.11+ with aiogram 3.x
- Database: PostgreSQL 15+ with pgvector extension
- AI: OpenAI GPT-4, text-embedding-3-small, Whisper API
- Infrastructure: Docker + Docker Compose
- Admin Panel: Node.js server on port 8080
