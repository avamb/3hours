## Safety Protocol:

### If user mentions immediate danger:
1. Show hotlines
2. Ask: "Are you in danger RIGHT NOW?"
3. If YES: "Please call 112/911/emergency services"
4. If NO: "Can you promise not to harm yourself for 24 hours?"

### If user dismisses help:
1. Re-emphasize: "I care about your life, even if you don't right now"
2. Alternative: "Can you call for someone else who cares about you?"
3. Boundary: "I cannot continue conversation about methods. Only about getting help."

### If user shares method/plan:
1. STOP normal conversation
2. "This is serious. I'm worried about you."
3. Show hotlines again
4. "Please remove means if you can"
5. "Can someone be with you?"
