# Daily Gratitude Prompts: Time-Aware & Personalized

## RU: Утренние вопросы (6:00-10:00)

### Pattern 1: Awakening Focus
```
Доброе утро! Что приятного было в твоем пробуждении сегодня?
```

**Variations:**
- "Каким хорошим моментом началось твое утро?"
- "Что порадовало тебя в первые минуты дня?"
- "Какое первое приятное ощущение ты заметил/заметила сегодня?"

**Use when:**
- Time: 6:00-10:00
- User typically responds in morning
- No previous morning question in last 3 days

---

### Pattern 2: Sensory Morning
```
Какой вкус или запах порадовал тебя за завтраком?
```

**Variations:**
- "Что вкусного было сегодня утром?"
- "Какой аромат утра тебе понравился?"
- "Что из утренних ритуалов принесло удовольствие?"

**Use when:**
- Time: 7:00-10:00
- User mentions food/coffee in history
- Sensory prompts work well for this user

---

### Pattern 3: Gentle Start
```
Доброе утро! Что хорошего ты заметил/заметила из окна утром?
```

**Variations:**
- "Какое первое доброе слово ты услышал/услышала сегодня?"
- "Что в утреннем ритуале тебя успокаивает?"
- "Чем порадовала тебя погода утром?"

**Use when:**
- Time: 6:00-9:00
- User is not a morning person (gentle approach)
- Previous responses show appreciation for nature

---

## RU: Дневные вопросы (10:00-17:00)

### Pattern 1: Social Interaction
```
Какой момент сегодня заставил тебя улыбнуться?
```

**Variations:**
- "Кто проявил к тебе доброту сегодня?"
- "Какой разговор сегодня был приятным?"
- "Кто порадовал тебя сегодня, пусть даже незнакомец?"

**Use when:**
- Time: 10:00-17:00
- User is typically social
- History shows appreciation for human connections

---

### Pattern 2: Achievement Focus
```
Что ты сегодня сделал/сделала хорошо, даже если это мелочь?
```

**Variations:**
- "Какое маленькое достижение у тебя было сегодня?"
- "В чем ты молодец сегодня?"
- "Что получилось лучше, чем ожидал/ожидала?"

**Use when:**
- Time: 12:00-17:00
- User struggles with self-worth
- Pattern of self-criticism in history

---

### Pattern 3: Mood Shift
```
Что сделало сегодняшний день чуть легче?
```

**Variations:**
- "Какой момент принес облегчение?"
- "Что помогло справиться с трудностями?"
- "Какая мелочь оказалась важной?"

**Use when:**
- Time: 11:00-16:00
- User mentioned stress/difficulty earlier
- Need for perspective shift
